#服务发现调用框架

##FEATURE

* 1.agent+javasist实现的代码注入功能 
* 2.kafka的异步消息  
* 3.AVRO RPC服现 
* 4.HTTP服务

##HOW TO RUN
* 1.git clone
* 2.mvn install 
* 3.cd ./soa_demo 
* 4.java -jar ./target/soa_demo.jar
